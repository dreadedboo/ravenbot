# self-hosted twitch bot written in python

## current general features:

- twitch bot
- json based configs
- custom commands and management
- livesplit integration
- modular, choose what to load on launch via modules.json

## current twitch features

- game and title commands
- livesplit commands (!pb, !bpt, !sob, !livesplit <game/title/setup>) NOTE: game title and setup only work with
  aop-runs' fork of livesplit at the moment
- followage command

## planned general features:
